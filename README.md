# Lume Store 

Sistema educacional desenvolvido em Python, inspirado em plataformas digitais de jogos, com funcionalidades de loja e venda de periféricos.

## Tecnologias utilizadas
- Python

##  Funcionalidades
- Sistema de login
- Catálogo de jogos e periféricos
- Carrinho de compras
- Pagamento
- Gestão de clientes
- Relatórios e vendas
- Área administrativa

##  Objetivo
Praticar lógica de programação, organização modular e desenvolvimento de sistemas em Python.
